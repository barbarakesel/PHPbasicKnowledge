# PHP Algorithms

### TASK1 Изучение алгоритмов сортировки

Реализуйте 3 самых популярных алгоритма сортировки. Обязательный вид сортировки - quick sort.
Для проверки работы алгоритмов, реализуйте клиентскую часть, задайте в ней несколько массивов и отсортируйте каждым алгоритмом. Результаты сортировки выведите в консоль для визуальной проверки.
</br>
### TASK2 Изучение алгоритмов поиска

Изучить алгоритмы поиска и реализовать линейный и бинарный поиск.
Для проверки работы алгоритмов, реализуйте клиентскую часть, задайте в ней несколько массивов и осуществите поиск каждым из алгоритмов. Результаты поиска выведите в консоль. Опционально можно вывести количество шагов, осуществлённых для поиска.
